# Quickstart

This guide shows how to clone, inspect, and run the public HomeLab AgentOps toolkit safely.

HomeLab AgentOps is a public and sanitized toolkit preview extracted from a private HomeLab operations project. It does not include private credentials, real tokens, production `.env` files, private logs, real backup archives, or sensitive infrastructure details.
